CASO DE USO: Pesquisar Obra
ATORES PRIMÁRIOS: Bibliotecário
ATORES SECUNDÁRIOS:
PROPÓSITO:      Bibliotecário buscar informações de uma obra
DESCRIÇÃO:      Este caso de uso se inicia quando o bibliotecário necessita buscar uma obra
REFERÊNCIAS:    
PRÉ CONDIÇÕES: 
FLUXO TÍPICO:   
                Bibliotecário                                   Sistema
                1-Entra no módulo de pesquisa
                                                                2-Pergunta se é livro ou periódico
                3-Responde
                                                                4-Pede dados para busca da obra
                                                                
                                                                5-Busca obra
                                                                
                                                                6-Se obra existe, retorna dados completos da obra                                   
                                                                
SEQUÊNCIA NÃO TÍPICA:
                Bibliotecário                                   Sistema
                1-Entra no módulo de pesquisa
                                                                2-Pergunta se é livro ou periódico
                3-Responde
                                                                4-Pede dados para busca da obra
                                                                
                                                                5-Busca obra
                                                                
                                                                6-Se obra não existe, retorna erro
